## 🌐 Vision
HelixAlpha is a decentralized biological research platform that replaces traditional "brute-force" drug discovery with **Autonomous Causal Reasoning**. By leveraging **Generative Flow Networks (GFlowNets)** and **Joint Embedding Predictive Architecture (JEPA)**, we simulate the "World Model" of biology to discover molecules that are not only effective but synthetically optimized.

## 🚀 Key Features

### 1. GFlowNet Causal Reasoning
Unlike traditional MCTS search trees, our **Generative Flow Network** samples diverse molecular candidates proportional to their biological reward. This allows the system to identify **causal synthesis paths**, determining the most efficient chemical trajectory to produce a drug.

### 2. JEPA World Model
We utilize **Joint Embedding Predictive Architecture** to predict latent biological states. This ensures our AI "understands" the rules of biology and avoids the "hallucinations" common in standard generative chemistry.

### 3. $HELIX Macroeconomics
A tri-token economy designed for long-term sustainability:
- **$HELIX Utility:** Powers the DePIN compute grid (Akash/Nosana).
- **IP-NFTs:** Fractionalized ownership of lead candidates ($C \geq 0.98$).
- **Proof-of-Improvement:** Rewards for researchers providing high-fidelity perturbation data.

### 4. Autonomous OpenClaw Agents
Our **OpenClaw** agents ("The Weavers") autonomously manage the "Loop until Perfect" method. They monitor simulation confidence, hire human Oracles for edge-case verification, and execute smart contracts.

## 🏗 Installation & Setup

1. **Clone the Repo:**
   ```bash
   git clone [https://github.com/nadhirhassen/helix-alpha.git](https://github.com/nadhirhassen/helix-alpha.git)
   cd helix-alpha
   ```

2. **Initialize OpenClaw Environment:**
   ```bash
   pip install openclaw-sdk rdkit-pypi torch-gfn
   ```

3. **Deploy Local Node:**
   ```bash
   openclaw start --skill brain/gfn_engine/trainer.py
   ```
